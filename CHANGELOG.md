# Changelog

All notable changes to this project will be documented in this file.

## [1.2.0] - 2025-01-24

### Added
- Implemented Material 3 Expressive design system throughout the app.
- New animated splash logo and improved app launch experience.
- Redesigned app widget to match the new Material 3 Expressive design.
- Added a metadata editor screen for editing song information.
- New playback speed adjustment (SpeedSheet) and enhanced sleep timer functionality.
- Added playlist import and export/share functionality.
- Added support for favoriting playlists.
- Integrated Deezer API for fetching artist pictures.
- Added a new Player widget for the home screen.
- Added option to stop playback by swiping the player.
- Added file sharing feature for songs.

### Changed
- Modernized app architecture and improved data layer reliability.
- Redesigned Navigation Bar and Navigation Drawer for better usability.
- Migrated media fetching mechanism to use WorkManager for improved performance.
- Refactored media player initialization and session management.
- Updated app icons and various UI components for a more consistent look.
- Updated numerous dependencies to their latest versions.

### Fixed
- Fixed an ANR (App Not Responding) issue during media loading.
- Fixed search bar behavior and expanded filter options.
- Fixed issues with screen rotation causing unnecessary media reloads.
- Fixed various UI layout and spacing issues across different screen sizes.
- Improved handling of window insets and system bar colors.
